### 自定义属性

- 自定义属性 shift+鼠标左键，弹出窗口中，选择“添加”，左侧填入属性，右侧填入属性值，即可实现相应效果
  - 自定义属性针对整个段落生效，不能针对段落中的部分文字生效
  - 属性可以叠加，中间用空格隔开即可（不需要加逗号，顿号等）
